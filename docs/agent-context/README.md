# Agent context — reference folder

Drop-in briefing docs for any agent picking up work on **Uniform
Logistic**. Each file is a self-contained snapshot of the project at a
point in time.

## Convention

- Files are named `NNNN-YYYY-MM-DD-slug.md`.
- `NNNN` is a zero-padded sequence. **The highest number is the newest
  / current context.**
- The table below is kept **newest-first** — the top row is always the
  latest.
- When you finish a meaningful chunk of work, add a NEW file (next
  number up) rather than rewriting the old one, and add a row to the
  top of the table. The history stays intact.

## How to brief a new agent (the "simple instruction")

> **"Read the latest file in `docs/agent-context/` (top row of its
> README) before doing anything."**

Equivalently: open the highest-numbered `NNNN-*.md` in this folder.
